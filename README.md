# style-transfer.tensorflow

This repository is a collection of various style-transfer models.


## Getting Started

### Prerequisites
* Python 3.6
* TensorFlow 1.10
* PIL: python image library
* numpy, matplotlib
* OS X and Linux (Not validated on Windows)
* Jupyter notebook
